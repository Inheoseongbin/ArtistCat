# Artist Cat

#### 교내 팀프로젝트 17개의 팀중 5등을 차지한 프로젝트입니다.
#### 제작기간 : 1~2개월
#### 게임 설명 : 스테이지 별로 공격해오는 몬스터들을 몬스터별로 가진 모형을 따라그려서 처치한다. 각 스테이지를 끝낼 때마다 나오는 강한 보스를 모두 처치해야 게임 클리어!
